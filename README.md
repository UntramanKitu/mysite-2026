# Http Status
- Pass 2xx Success 
- 200 OK สำเร็จ
- 201 Created สร้างใหม่ 
- Pass 3xx Redirection 
- 301 Move Permenent ย้ายถาวร
- 302 Move Temporaly ย้ายชั่วคราว 
- Fail 4xx Error from Client
- 400 Bad Request คำขอไม่ถูกต้อง
- 401 Unauthorized ไม่ได้รับอนุญาติ
- 403 Forbidden ห้ามเข้าถึง
- 404 Not Found ไม่พบ 
- 405 Method Not Allowed ใช้ method ไม่ได้ 
- Fail 5xx Error from Server
- 500 Internal Server Error เซิฟเวอมีปัญหา
- 502 Bad Gateway เซิร์ฟเวอร์ต้นทางได้รับคำตอบที่ผิดพลาด
- 503 Service Unavailable เซิร์ฟเวอร์ปิดปรับปรุงชั่วคราว
- 504 Gateway Timeout หมดเวลาเชื่อมต่อ
